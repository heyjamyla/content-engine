One-time Setup · Google Sheets Sync

Connect your Sheet.

Five minutes once. Then every Sacral Hit you log — on your phone, laptop, anywhere — lands in the same Google Sheet.

Step 01
Create your Google Sheet
Go to sheets.new to create a new spreadsheet. Name it something like MG Content Engine — Sacral Hits.

In Row 1, add these exact headers in columns A–E:
Timestamp Brand Hit Gut Notes
Step 02
Open Apps Script
Inside your new Sheet, go to Extensions → Apps Script. Delete any existing code in the editor, then paste in the code below exactly as written:
function doPost(e) { try { var sheet = SpreadsheetApp .getActiveSpreadsheet() .getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date().toLocaleString(), data.brand || '', data.hit || '', data.gut || '', data.notes || '' ]); return ContentService .createTextOutput(JSON.stringify({ status: 'ok' })) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ status: 'error', message: err.message })) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { var sheet = SpreadsheetApp .getActiveSpreadsheet() .getActiveSheet(); var rows = sheet.getDataRange().getValues(); var hits = rows.slice(1).map(function(r) { return { timestamp: r[0], brand: r[1], hit: r[2], gut: r[3], notes: r[4] }; }); return ContentService .createTextOutput(JSON.stringify({ hits: hits })) .setMimeType(ContentService.MimeType.JSON); }
Step 03
Deploy as a Web App
In the Apps Script editor:

1. Click Deploy → New deployment
2. Click the gear icon next to "Type" and select Web app
3. Set Execute as: Me
4. Set Who has access: Anyone
5. Click Deploy and authorize when prompted
6. Copy the Web app URL — it starts with https://script.google.com/macros/s/...
💡 Every time you edit the script you'll need to create a new deployment — but you only do this setup once.
Step 04
Paste your Web App URL here
Paste the URL from Step 03 below and click Connect. That's it — all your Sacral Hits will start syncing to your Sheet from any device.
