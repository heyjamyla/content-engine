Weekly Rhythm · MG Edition

Low lift.
High truth.

MGs aren't built for daily content grind. This is a rhythm your energy can actually sustain.

Tap a day to expand
Monday
Open eyes. Log what's alive.
▼
  • Review any captures from the weekend
  • 2 min: what's a question I keep getting asked?
  • Log anything that got your attention — don't filter it
  • Don't plan this week's content yet. Just gather.
Tuesday
Capture + continue.
▼
  • Log captures from the production floor, customer interactions, your own rituals
  • Optional: voice memo while driving, walking, in the lab
  • Light engagement: respond to comments as content research
Wednesday
The Burst Session.
▼
  • 60–90 min protected time — the only non-negotiable day
  • Review captures, pick 2–3 that still feel alive
  • Film back-to-back — talking points only, no full script
  • One video per brand, or one cross-brand piece — follow the energy
  • Don't edit today. Just capture.
Thursday
Splinter the burst.
▼
  • Pull the 2–3 best clips or moments
  • Write or voice-memo 3 Threads posts from the same energy
  • Draft one email hook for Oyin or Hey Jamyla
  • Identify 1–2 short-form clips for Reels/TikTok
  • Bank what's not ready — it'll keep
Friday
Publish what's ready.
▼
  • Post or schedule anything that feels done
  • "Done" = good enough to be useful. Not perfect.
  • If it still feels alive, post it. If the energy's gone, archive it.
  • Mark posted in the Review tab
Weekend
Rest is strategy.
▼
  • No content work. Energy needs to recharge.
  • If a spark comes — voice memo it, don't act on it
  • Yoga, nature walks, music, ritual — this is how you fill the well
  • Your best Monday captures come from a real weekend

One-time Setup · Google Sheets Sync

Connect your Sheet.

Five minutes once. Every capture syncs from any device. Your sisters each do their own setup — fully independent, each locked with their own passphrase.

Step 01
Create your Google Sheet
Go to sheets.new. Name it MG Content Engine.

Create two tabs: rename Sheet1 to Captures, add a second called Posted.

Captures — Row 1 headers (columns A–E):
Timestamp Brand Capture Gut Notes


Posted — Row 1 headers (columns A–I):
Date Posted Brand Original Capture AI Bullets Brand Angles Camera Prompt Your Note Post URL Original Timestamp
Step 02
Add the Apps Script
Go to Extensions → Apps Script. Delete existing code and paste this — replace YOUR_PASSPHRASE with your secret word before deploying:
var SECRET = 'YOUR_PASSPHRASE'; function doPost(e) { try { var data = JSON.parse(e.postData.contents); if (data.token !== SECRET) return respond({status:'unauthorized'}); var ss = SpreadsheetApp.getActiveSpreadsheet(); if (data.action === 'post') { ss.getSheetByName('Posted').appendRow([ new Date().toLocaleString(), data.brand||'', data.hit||'', data.bullets||'', data.brandAngles||'', data.cameraPrompt||'', data.note||'', data.url||'', data.originalTimestamp||'' ]); } else { ss.getSheetByName('Captures').appendRow([ new Date().toLocaleString(), data.brand||'', data.hit||'', data.gut||'', data.notes||'' ]); } return respond({status:'ok'}); } catch(err) { return respond({status:'error',message:err.message}); } } function doGet(e) { if (e.parameter.token !== SECRET) return respond({status:'unauthorized'}); var tab = e.parameter.tab || 'Captures'; var rows = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(tab).getDataRange().getValues(); var hits = rows.slice(1).map(function(r){ return tab==='Posted' ? {datePosted:r[0],brand:r[1],hit:r[2],bullets:r[3], brandAngles:r[4],cameraPrompt:r[5],note:r[6],url:r[7],originalTimestamp:r[8]} : {timestamp:r[0],brand:r[1],hit:r[2],gut:r[3],notes:r[4]}; }); return respond({hits:hits}); } function respond(obj){ return ContentService.createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }
Step 03
Deploy as a Web App
1. Click Deploy → New deployment
2. Gear icon → Web app
3. Execute as: Me  ·  Who has access: Anyone
4. Click Deploy, authorize, copy the Web app URL
💡Edit the script? Create a new deployment. The URL changes — update it in Step 04.
Step 04
Connect your Sheet
Paste your Web App URL and the passphrase you used in the script. This locks your Sheet — only your copy of this tool can write to it.

🔑Must match exactly what you put in var SECRET in Step 02.
